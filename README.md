# JGlo

JGlo is a Intellij plugin that can interact with Glo boards.

It's objective is to bring Glo Board operations closer to you, so that "alt + tab" is no longer used.



# Roadmap

Basic operations:
+ [x] Authentication
+ [x] Create cards
+ [x] Edit cards
+ [x] Delete cards
+ [ ] Create columns
+ [ ] Assignees
+ [ ] Due date


Extra features:
+ [ ] Time tracking
+ [ ] Images
+ [ ] Labels
+ [ ] Notifications
